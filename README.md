An end-to-end Excel data pipeline and interactive dashboard analyzing NSE stock trends using Power Query and Linear Regression for investment decision support.
# NSE Stock Market Analysis (Dec 2025 – Jan 2026)

## 📌 Project Overview
This project involves an end-to-end data analytics pipeline built entirely in Microsoft Excel. The goal was to analyze historical stock market data from the National Stock Exchange (NSE) to support investment decision-making, mitigate risk, and optimize returns. 

By transforming raw, unstructured CSV data into an interactive dashboard, this project provides stakeholders with a clear, decision-support tool to identify market trends and anticipate price movements.

## 🛠 Tools & Technologies Used
* **Microsoft Excel:** Primary analytical engine.
* **Power Query:** Used for Data Extraction, Transformation, and Loading (ETL).
* **Pivot Tables & Pivot Charts:** Used for Exploratory Data Analysis (EDA) and data aggregation.
* **Excel Analysis ToolPak:** Used for statistical modeling (Linear Regression).

## 🚀 Methodology

### 1. Data Cleaning & Preparation (ETL)
* Extracted raw daily trading data (Open, High, Low, Close, Volume) for major NSE stocks.
* Utilized **Power Query** to automate the cleaning process, standardize date formats, and calculate daily price fluctuations.

### 2. Exploratory Data Analysis (EDA)
* Built **Pivot Tables** to aggregate monthly metrics and calculate moving averages.
* Analyzed trading volumes across different stocks to identify liquidity and market interest.

### 3. Statistical Modeling
* Conducted **Linear Regression** analysis to evaluate the mathematical relationship between independent variables (Trading Volume) and dependent variables (Closing Price). 
* Used R-Square and P-value outputs to assess market volatility and stock stability for risk mitigation.

### 4. Interactive Dashboard
* Designed a front-end visualization board using Pivot Charts.
* Integrated **Slicers** to allow users to dynamically filter data by specific stock tickers (e.g., RELIANCE, TCS, HDFCBANK) and timeframes, improving data transparency.

## 💡 Key Learnings & Business Impact
This project demonstrates that complex financial data does not always require heavy programming languages to yield actionable insights. By leveraging advanced Excel functionalities, I built an automated, user-friendly tool that allows non-technical stakeholders to visualize risk and make data-backed investment choices.
